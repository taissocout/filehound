# FileHound

OSINT file discovery + metadata intelligence (ExifTool) with Markdown write-up.

## Features
- Multi-engine dork search (Brave/Bing/DDG/Yandex/etc.)
- Downloads only real files (Content-Type validation)
- Extracts metadata (txt + json)
- Generates professional Markdown report

## Requirements
- curl, exiftool, file, jq (optional)

Install (Kali/Debian):
```bash
sudo apt update && sudo apt install -y curl exiftool file jq
